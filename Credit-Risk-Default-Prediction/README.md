# Credit Risk Default Prediction

A binary classification project to predict loan default risk using 
customer, loan, and credit bureau data.

## Overview

Built a machine learning pipeline to identify high-risk borrowers from a merged 
dataset of 50,000+ records across three data sources. The project covers the full 
modeling lifecycle. From data cleaning and feature engineering to model training, 
hyperparameter tuning, and a Streamlit deployment.
